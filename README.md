## Installation

UP AND RUNNING
Install dependencies:

If you use yarn:

$ yarn
Else if you use npm:

$ npm install
Start the development server
$ yarn start
or

$ npm start
Testing
$ yarn test
Or

$ npm test
Production Build
$ yarn build
or

$ npm run build
